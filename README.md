# medical-record-system

## Description

This is a electronic health record system project. 

It uses Node.js for the backend and React for the Frontend

## Prerequisites
- node
- mysql

## Libraries used
### Backend
- express
- esm
- cors
- bcrypt
- jsonwebtoken
- mysql

### Frontend
- axios
- styled-components
- react-router-dom
- @devexpress/dx-react-scheduler
